# Forward Together - Therapy Website

A Next.js website for Forward Together therapy practice, offering compassionate counseling for women at every life stage.

## Tech Stack

- **Framework:** Next.js 14.2
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Deployment:** Vercel

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

### Build

```bash
npm run build
npm start
```

## Project Structure

